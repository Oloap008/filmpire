# [Filmpire](https://filmpire-liart-eight.vercel.app/)

A personalised application where you can search, watchlist, and favorite movies.

## Features

- Browse movies by
  - Popularity
  - Rating
  - Upcommming
  - Genre
- Favorite and Watchlist movies

## Technology Stack

- React
- React Router
- Redux Toolkit
- RTK Query
- Material UI
- Alan AI
- TMDB API
